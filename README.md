# OAuth 2.0 with Kotlin
This is a personal project to practice with the following tecnologies:
 - OAuth 2.0
 - Open ID Connect
 - Kotlin

## Running the application
 - download and build the project with maven
 - spring-boot:run

## Result
### 1. Go to URL
http://localhost:8080/user/getUsersList
